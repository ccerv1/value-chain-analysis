    Lecture Slides¶

    Slide decks from the Value Chain Analysis guest lecture at Columbia SIPA, taught annually since 2017. Each year folder contains the .pptx source and any PDF exports or supporting data.

    By Year¶

    2025 — Current¶

    • Value Chain Analysis.pptx — Single combined deck. Covers Vietnam and Rwanda as primary cases. Introduces the "Map / Breakdown / Benchmark" framework.
    • slide-data.xlsx — Data workbook behind the charts (prices, yields, farmer shares)

    2024¶

    • Value Chain Analysis.pptx — Single combined deck. Same structure as 2025 with 2024 market data.
    • slide-data.xlsx — Data workbook

    2023¶

    • Day 1.pptx / Day 1.pdf — Coffee 101, mapping, and breakdown
    • Day 2.pptx — Benchmarking, recommendations, and country deep dives

    2022¶

    • Day 1.pptx / Day 1.pdf — Condensed to a single session

    2021¶

    • Day 1.pptx / Day 1.pdf — Coffee 101, VCA framework
    • Day 2.pptx / Day 2.pdf — Country cases and recommendations

    2020¶

    • Day 1.pptx / Day 1.pdf — First COVID-era offering
    • Day 2.pptx / Day 2.pdf

    2019¶

    • Day 1.pptx / Day 1.pdf
    • Day 2.pptx / Day 2.pdf

    2018¶

    • Day 1.pptx / Day 2.pptx / Day 2.pdf — Individual session decks
    • All Slides.pptx / All Slides.pdf — Combined deck
    • Value Chain Analysis.pptx — Earlier draft or alternate version

    2017 — First Year¶

    • Day 1.pptx / Day 1.pdf / Day 2.pptx — Session decks
    • Value Chain Analysis.pptx — Combined deck
    • colombia-slide-deck.pptx — Standalone Colombia case study deck
    • agenda.docx — Session agenda
    • slides-combined.pdf — PDF export of all slides

    Notes¶

    • The course shifted from a two-day format (Day 1 + Day 2) to a single combined deck starting in 2024.
    • Slide content evolves year over year but the core framework (Map / Breakdown / Benchmark) has been stable since 2017.
    • For the written version of the lecture content, see lecture-notes/ and skills/.
