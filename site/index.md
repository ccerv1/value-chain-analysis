# Value Chain Analysis

![Value Chain Analysis](photos/vca-hero.png)

Carl Cervone's guest lecture materials for Value Chain Analysis, presented as part of Professor Glenn Denning's Global Food Systems course at Columbia University's School of International and Public Affairs ([SIPA](https://sipa.columbia.edu/)). 

The talk provides a framework for analyzing value chains, using the coffee value chain as the primary example.

The current version of this site reflects content from the March 2026 lecture. You can also browse all [past lecture slides](lectures/README.md) (2017-2025).
