# Value Chain Analysis

![Value Chain Analysis](photos/vca-hero.png)

Guest lecture materials for Professor Glenn Denning's course at Columbia SIPA. This site covers how to trace the flow of value from farm to cup, diagnose where the problems are, and turn analysis into actionable recommendations.

The current version reflects the March 2026 lecture. You can also browse all [past lecture slides](lectures/README.md) (2017-2025).
