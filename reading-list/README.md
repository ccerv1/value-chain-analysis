# Reading List

Background readings, case study source documents, and reference materials for the Value Chain Analysis course.

## Case Studies

Primary source documents for the country cases covered in the lecture.

| Reading | Description |
|---------|-------------|
| [The Coffee Crisis (2003)](https://github.com/ccerv1/value-chain-analysis/raw/main/reading-list/case-studies/coffee-crisis-2003.pdf) | Overview of the 2001-2003 global coffee crisis — price collapse, farmer impact, structural causes |
| [Vietnam Middlemen (2013)](https://github.com/ccerv1/value-chain-analysis/raw/main/reading-list/case-studies/vietnam-middlemen-2013.pdf) | Enveritas report on the role of middlemen in Vietnam's coffee supply chain. Key source for the 95% farmer share finding. |
| [Colombia (2014)](https://github.com/ccerv1/value-chain-analysis/raw/main/reading-list/case-studies/colombia-2014.pdf) | Colombia coffee sector analysis — FNC model, cost pressures, government support |
| [Ethiopia: ECX and Specialty Coffee](https://github.com/ccerv1/value-chain-analysis/raw/main/reading-list/case-studies/ethiopia-ecx-and-specialty-coffee.pdf) | How the Ethiopia Commodity Exchange affected traceability and specialty premiums |
| [East Africa (2007)](https://github.com/ccerv1/value-chain-analysis/raw/main/reading-list/case-studies/east-africa-2007.pdf) | East African coffee value chain overview (Rwanda, Kenya, Tanzania, Ethiopia) |
| [Colombia Slide Deck](https://github.com/ccerv1/value-chain-analysis/raw/main/reading-list/case-studies/colombia-slide-deck.pdf) | Standalone Colombia presentation from the 2017 lecture |

## Reference Guides

Methodological frameworks and sector overviews.

| Reading | Description |
|---------|-------------|
| [Coffee 101](https://github.com/ccerv1/value-chain-analysis/raw/main/reading-list/reference-guides/coffee-101.pdf) | Comprehensive introduction to the global coffee industry — production, processing, trade |
| [WFP Value Chain Analysis Guide](https://github.com/ccerv1/value-chain-analysis/raw/main/reading-list/reference-guides/wfp-value-chain-analysis-guide.pdf) | WFP's practical guide to conducting value chain analysis in development contexts |
| [USAID Market Systems Framework](https://github.com/ccerv1/value-chain-analysis/raw/main/reading-list/reference-guides/usaid-market-systems-framework.pdf) | USAID's market systems development framework |
| [USAID WEAI](https://github.com/ccerv1/value-chain-analysis/raw/main/reading-list/reference-guides/usaid-weai.pdf) | Women's Empowerment in Agriculture Index — methodology and application |
| [USAID Globalization and the Small Firm](https://github.com/ccerv1/value-chain-analysis/raw/main/reading-list/reference-guides/usaid-globalization-and-the-small-firm.pdf) | How globalization affects small agricultural producers |
| [Economic Viability and Sustainability of Coffee Production](https://github.com/ccerv1/value-chain-analysis/raw/main/reading-list/reference-guides/economic-viability-sustainability-coffee-production.pdf) | Analysis of coffee production economics and sustainability challenges |
