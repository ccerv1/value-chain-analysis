# Reading List

Background readings, case study source documents, and reference materials for the Value Chain Analysis course.

## Case Studies (`case-studies/`)

Primary source documents for the country cases covered in the lecture.

| File | Description |
|------|-------------|
| `coffee-crisis-2003.pdf` | Overview of the 2001-2003 global coffee crisis — price collapse, farmer impact, structural causes |
| `vietnam-middlemen-2013.pdf` | Enveritas report on the role of middlemen in Vietnam's coffee supply chain. Key source for the 95% farmer share finding. |
| `colombia-2014.pdf` | Colombia coffee sector analysis — FNC model, cost pressures, government support |
| `ethiopia-ecx-and-specialty-coffee.pdf` | How the Ethiopia Commodity Exchange affected traceability and specialty premiums |
| `east-africa-2007.pdf` / `.doc` | East African coffee value chain overview (Rwanda, Kenya, Tanzania, Ethiopia) |
| `colombia-slide-deck.pdf` | Standalone Colombia presentation from the 2017 lecture |
